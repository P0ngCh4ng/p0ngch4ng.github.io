【問5】次に示す関数addを定義せずに、呼び出し側で無名関数にインライン展開してください。

f g = g 1 2
add x y = x + y

解答

main = do
    print $ (\g -> g 1 2) (\x -> \y -> x + y)
    print $ (\x -> \y -> x + y) 1 2
