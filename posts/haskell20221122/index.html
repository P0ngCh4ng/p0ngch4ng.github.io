問題6

次に示す関数をラムダ式に置き換える

fact n | n > 0 = n * fact (n - 1)

main = do
    print $ fact 5
	```
	
	回答
	```
	fact = \n -> case n ofM\
	0 -> 1
	_ | n > 0 -> n * fact (n - 1)

main = do
    print $ fact 5
	```


## 問題７
次に示す関数addをラムダ式で書き換えてください。

add :: Int -> Int -> Int add x y = x + y

main = do print $ add 2 3

回答

add :: Int -> Int -> Int add = \x -> \y -> x + y




## 問題８
次に示す関数addを定義せずに、呼び出し側で無名関数にインライン展開してください。

add x y = x + y

main = do print $ add 2 3 ```

回答

print $ ( \x -> \y -> x + y) 2 3
